# TrustChain Certificate – Fake Certificate Detection (EdTech Project)

## About the Project
TrustChain Certificate is a blockchain system to stop fake certificates. Institutes can issue certificates securely, and anyone can check them instantly.

## Project Structure
- backend → Node.js + MongoDB + Smart Contract
- frontend → React.js UI for students, institutes, verifiers

## Features
- Blockchain-based certificate storage
- Prevent fake certificates
- Secure login
- Institute dashboard
- Students can access/verify certificates
- Public verification portal

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Blockchain: Solidity + Smart Contract
- Tools: GitHub, MetaMask
